# Python 课程大作业：A4 复印纸价格与采购模拟

本项目为课程大作业，研究对象为 70g、每箱 5 包、每包 500 张规格的 A4 复印纸。整体工作包括：从京东获取符合条件的商品信息，通过慢慢买接口采集历史价格数据，在此基础上进行统计分析与 Dash 可视化展示，并实现基于预算与库存约束的采购数量模拟。仓库内数据与结论仅供课程演示，不构成任何商业或投资建议。

## 项目内容说明

京东列表采集脚本 京东_爬虫.py 基于 Selenium 访问京东站点，需用户手动完成登录。搜索关键词为「70g A4复印纸 每箱5包 每包500张」，列表按页抓取时由用户翻页后确认，程序解析标题中同时包含 70g、A4、复印纸、5包、500张 的商品，结果输出为 jd_a4_paper_filtered_manual.csv。

慢慢买历史价格脚本 慢慢买_爬虫.py 将打开历史低价查询页面，用户在终端粘贴商品页完整链接，程序请求相应接口并将数据按商品保存至 data 目录，文件名为经清理的商品标题。

交互式图表脚本 交互式图表.py 读取 analysis 目录下的汇总统计与低价相关表格，使用 Dash 与 Plotly 展示图表。箱线图数据来源于 cleaned 目录中各商品的 cleaned 后缀 CSV。第二个选项卡中的价格趋势图与 FFT 频谱图路径为 analysis 下各商品子目录中的 price_trend.png 与 fourier_spectrum.png；若仓库中未包含上述图像文件，该部分可能无法显示，需自行生成并放置到对应路径。

采购决策脚本 决策.py 读取 analysis 目录中的 daily_average_price_all_products.csv（全商品日均价序列）。用户在终端按「月.日,箱价,品牌」格式逐条输入采购记录，至少输入 8 条后方可结束。程序设定初始库存 200 包、总预算 20000 元，并按月份规则扣减用纸量，输出写入 input_data.csv 与 output_data.csv。日期解析在源码中按 2025 年处理，若需其他年份须在代码中修改。

仓库中的 data、cleaned、analysis 目录包含课程实验过程中生成的示例数据与分商品分析结果，便于在未重新爬取的情况下复现实验与演示。

## 运行环境

京东采集脚本依赖本机安装 Chrome 浏览器及与之匹配的 WebDriver，具体配置以本机 Selenium 环境为准。

Python 第三方依赖见项目根目录下的 requirements.txt。安装命令如下。

    pip install -r requirements.txt

慢慢买脚本中为便于调试关闭了部分 SSL 证书校验相关警告，该写法仅适用于课程环境，不建议用于生产系统。

## 运行方式

在项目根目录下依次或按需执行：

    python 京东_爬虫.py
    python 慢慢买_爬虫.py
    python 交互式图表.py
    python 决策.py

执行 交互式图表.py 后，在浏览器中访问终端提示的本地地址（一般为 http://127.0.0.1:8050）即可打开仪表盘。

脚本内数据路径均相对于各脚本文件所在目录解析，将项目整体复制至其他路径后一般无需修改盘符或绝对路径。

## 其他说明

电商页面与第三方接口可能变更，采集脚本不保证长期可用，若失效需自行调整选择器或请求参数。采购数量与价格相关逻辑均为课程设计中的简化模型，不代表真实采购策略。
